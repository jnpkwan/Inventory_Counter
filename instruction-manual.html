Ano ang app na ito?
Overview — basahin mo muna ito!

Ang AcaciaHBC Inventory App ay ginagamit para mag-count ng mga produkto sa isang tindahan. Bilhin mo lang ang bilang ng bawat SKU, i-record ang may sira, tapos i-export bilang Excel file — tapos na! 🎉

1
🏠 Punan ang iyong impormasyon
Pangalan mo at kung saang tindahan ka.
2
📦 I-count ang mga produkto
Hanapin ang bawat SKU, ilagay ang bilang.
3
⚠️ I-record ang mga sira
Kung may damaged na produkto, ilagay ito.
4
📊 I-review at i-export
Tingnan ang summary, tapos i-download ang Excel file.
📱
Gumagana ito sa phone!
Ang app na ito ay ginawa para sa mobile. Gamitin mo ito sa iyong smartphone para mas madali mag-count sa loob ng tindahan.
📶
Internet — kailangan sa simula at sa pagpapadala lang!
Sa simula: Kailangan ng internet nang buksan mo ang app para ma-download ang listahan ng mga produkto, stores, at system count data. Kapag na-load na ito, pwede ka nang mag-count kahit walang signal — offline na ang lahat ng counting at encoding.

Sa pagpapadala: Kailangan ulit ng internet kapag ise-share mo ang file via Messenger o ibang app.

I-tap ang bawat tab para lumipat sa ibang parte ng app. Sundan mo ang order — 1, 2, 3, 4!

1
Store Info — Punan ang Iyong Detalye
Tab 1 · Gawin ito MUNA bago mag-count
👋
Ito ang unang gagawin mo!
Kapag binuksan mo ang app, dito ka darating. Kailangan mong punan ito bago ka makapag-start ng counting.
1
Ilagay ang iyong pangalan
Sa field na Merchandiser Name, i-type ang iyong buong pangalan. Halimbawa: "Juan dela Cruz"
2
Piliin ang tindahan
Sa field na Location, i-tap ang dropdown menu at piliin ang pangalan ng tindahan kung saan ka nag-icount ngayon.
3
Count Period at Date/Time — automatic na!
Hindi mo na kailangang baguhin ito. Awtomatiko itong naka-fill ng app. ✅
4
I-tap ang "Save & Start Counting →"
Kapag puno na ang lahat, i-tap ang Save & Start Counting → na berdeng button. Lilipat ka na sa Count tab!

2
Count — Bilang ang mga Produkto
Tab 2 · Dito mo ida-dagdag ang bawat SKU
📦
Ano ang mangyayari dito?
Sa tab na ito, hahanapin mo ang bawat produkto, ilalagay ang bilang ng units na nakita mo sa tindahan, tapos ida-dagdag sa iyong listahan.
🔍 Paano mag-search ng produkto
1
I-tap ang search box
Makikita mo ang box na nagsasabing "Search products by name or SKU…". I-tap ito para mag-type.
2
I-type ang pangalan o SKU
Halimbawa, i-type mo ang "drill" o ang SKU code ng produkto. Makikita mo ang listahan ng mga produkto na lumalabas habang nagta-type ka.
3
I-tap ang produkto
Makikita mo ang mga resulta na may pangalan ng produkto at SKU code nito. I-tap ang tamang produkto para mapili ito. Lalabas ang berdeng bar na nagpapakita ng napiling produkto.
🔢 Paano ilagay ang bilang (Qty)
1
Gamitin ang + at − na buttons
May malaking − at + na button sa magkabilang side. I-tap ang + para dagdagan ang bilang, at − para bawasan.
2
O i-type mismo ang bilang
I-tap ang numero sa gitna at direktang i-type ang bilang. Halimbawa, kung 24 units, i-type mo ang 24.
3
I-tap ang "Add to Count"
Kapag tama na ang bilang, i-tap ang berdeng Add to Count button. Makikita mo ang produkto sa listahan sa ibaba! 🎉
💡
May "System Count" na lalabas!
Kapag pinili mo ang isang produkto, makikita mo ang bilang mula sa sistema (System Count). Ito ang reference mo — kung malayo ang iyong count doon, baka may mali. Double-check lang!
📋 Ang iyong listahan ng Counted Items
👀
Makikita mo ang lahat ng na-count mo
Sa ibaba ng Add Item section, may listahan ng lahat ng produkto na naidagdag mo na. May pangalan, SKU, at bilang (units). Ang malaking berdeng numero sa kanan = ang count mo.
✏️
Nagkamali sa count? I-delete muna, tapos idagdag ulit
Hindi pwedeng direktang baguhin ang count ng isang item na naidagdag na. Kung may mali ka, i-tap ang ✕ button para i-delete ang item, tapos hanapin ulit at idagdag ng tamang bilang.
🗑
Paano mag-delete ng item
May maliit na ✕ na button sa tabi ng bawat item. I-tap ito para ma-remove ang item sa listahan. Magtanong muna bago i-tap — walang undo!
🚨
MAHALAGA: Lagyan ng ZERO ang walang stock!
Kung ang isang SKU ay nasa tindahan pero wala kang makitang stock — huwag itong laktawan! Kailangan mo pa rin itong hanapin at idagdag na may bilang na 0.

Bakit? Para malaman ng sistema na tiningnan mo ito — hindi lang nakalimutan o nalaktawan. Ang produktong wala sa iyong listahan ay mukhang hindi na-count, kahit wala talagang stock.
Halimbawa: May curtain rod na dapat nasa shelf, pero empty ang shelf ngayon. Hanapin ang SKU → ilagay ang qty na 0 → i-tap ang Add to Count. ✅
💾
Auto-saved ang lahat!
Kahit isara mo ang app, nandoon pa rin ang iyong data kapag bumalik ka. Automatic itong nase-save sa iyong phone. Hindi ka dapat matakot na mawala ang iyong trabaho. Napakaliit din ng data na nase-save — hindi ito kukuha ng malaking space sa iyong phone.
🔴
Mag-ingat sa "Clear All Data"!
May pulang button sa ibaba na 🗑 Clear All Data. Huwag itong i-tap maliban kung talagang gusto mong burahin ang lahat ng nai-count mo. Hindi ito marecover!
3
Damages — I-record ang mga Sira
Tab 3 · Para sa mga damaged na produkto
ℹ️
Kailan gagamitin ang tab na ito?
Kapag nakakita ka ng produkto na sira, nabasa, nabali, o may depekto ang packaging — ilagay ito dito. Hiwalay ito sa regular na count.
1
I-tap ang "3 · Damages" tab
Pumunta sa Damages tab sa ibaba ng screen.
2
Hanapin ang produkto
Gamitin ang search box — i-type ang pangalan o SKU ng siraing produkto. I-tap ang tamang produkto mula sa listahan.
3
Ilagay ang bilang ng damaged units
Gamitin ang + at − para ilagay kung ilang piraso ang sira. Default ay 1 — baguhin kung kailangan.
4
I-type ang deskripsyon ng sira
Sa Damage Description box, ilagay kung bakit sira. Halimbawa: "Broken corner", "Nabasa", "Dented packaging", "Natanggal ang label".
5
I-tap ang "Record Damage"
I-tap ang berdeng Record Damage button. Lalabas ang damaged item sa listahan sa ibaba. ✅
📝
Maging detalyado sa description!
Mas maganda kung detalyado ang isinulat mo — para alam ng manager kung ano talaga ang nangyari sa produkto. Halimbawa: "Sira ang takip, tumagas ang laman" ay mas maganda kaysa sa "sira" lang.
4
Review & Export — Tapusin na!
Tab 4 · I-check ang lahat bago mag-submit
✅
Ito ang huling hakbang!
Dito mo makikita ang buong summary ng lahat ng na-count mo at mga damages. Kapag tama na lahat, mag-e-export ka ng Excel file.
📋 Paano mag-review
1
I-tap ang "4 · Review" tab
Pumunta sa Review tab sa ibaba ng screen. Makikita mo ang lahat ng impormasyon.
2
Tingnan ang Store Info
I-check kung tama ang iyong pangalan, store, at count period. Kung may mali — bumalik sa Tab 1 at baguhin.
3
Tingnan ang Count Summary
Makikita mo ang Total SKUs Counted at Total Categories. Tama ba ang bilang? Kumpleto ba ang lahat ng produkto na dapat mong bilangin?
4
Tingnan ang Damage Summary
Makikita mo rin ang buong listahan ng mga damaged items na na-record mo.
📊 Paano mag-export ng Excel
1
I-tap ang "Export & Save Excel" button
Makikita mo ang malaking berdeng button na ↑ Export & Save Excel. I-tap ito.
2
Lalabas ang confirmation window
May lilitaw na preview ng iyong data. I-tap ang Let's Go → para ituloy ang export.
3
Na-download na ang file! ✅
Awtomatikong dina-download ang Excel file sa iyong phone. Ang pangalan nito ay katulad nito: IC_SM Mall of Asia_28Mar2026_1045.xlsx
📤 Paano i-send ang file
A
Gamit ang Share button (mas madali!)
Kapag na-download na, lalabas ang 📤 Share via Messenger button. I-tap ito at piliin kung saan mo gustong i-send — Messenger, Viber, Email, etc.
B
O manual na i-attach
Buksan ang iyong email o messaging app → I-tap ang 📎 Attach → Hanapin ang file sa Downloads folder → I-send. Ganun lang!
📱
Hanapin ang file sa Downloads
Ang Excel file ay nandoon sa Downloads folder ng iyong phone. Buksan ang Files app (o My Files) → Downloads → hanapin ang file na nagsisimula sa "IC_".
💡
Mga Importanteng Tips
Basahin mo rin ito — helpful ito!
📶
I-connect sa internet MUNA bago pumasok sa tindahan!
Bago ka pumunta sa store, buksan ang app habang may internet ka pa. Kailangan nito ng koneksyon sa simula para ma-download ang mga produkto, listahan ng stores, at system count. Kapag na-load na — offline ka na pwedeng mag-count nang walang signal. Huwag kalimutang mag-connect ulit kapag magse-share na ng file.
🔋
I-charge ang phone bago mag-count!
Matagal ang inventory counting. Siguraduhing may sapat na baterya ang iyong phone bago ka magsimula.
🔄
Pwede kang bumalik sa app anytime!
Kahit isara mo ang app at bumalik mamaya, nandoon pa rin ang lahat ng na-count mo. Auto-saved ang lahat sa iyong phone.
📅
20 araw lang ang buhay ng data!
Awtomatikong nababura ang data pagkalipas ng 20 araw mula noong huli kang nag-export. Kaya siguraduhing nai-export mo na ang file bago lumipas ang 20 araw!
🚫
Huwag i-tap ang "Clear All Data" nang walang dahilan!
Ang pula na 🗑 Clear All Data button ay permanenteng nagbubura ng lahat ng iyong nai-count. Walang undo. I-tap lang ito kung talaga kang nagsisimula ng bagong count at sigurado kang tapos na ang dati.
🔍
May Category Filter!
Sa Count page, makikita mo ang mga colored chips sa itaas ng search bar (halimbawa: "All", "Padlocks", "Curtain Accessories"). I-tap ang isang category para ma-filter ang listahan at mas madaling mahanap ang produkto.
